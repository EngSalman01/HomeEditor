Getting Started

Draw Walls

Click on wall tool to build outer/main walls. Click and drag to create wall segments.

Create Partitions

Use the partition tool to create inner rooms and divide spaces.

Add Elements

Place doors, windows, and other elements using the tools on the left sidebar.

Select and Modify

Use the selection tool to move, resize, or delete elements in your design.
